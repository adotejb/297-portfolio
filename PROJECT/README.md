<h1>CHIP 690-297 PROJECT - Pharmacy Sales Tracker</h1>

<p>
	The purpose of this application is to assist a pharmacy in tracking sales for diabetes and hypertension medication over a 5-year period (2020 - 2025). Two fitting drugs were pulled from a Smarthealth FHIR API to demonstrate the application's functions.
</p>

<h2>Medication Options</h2>

<p>
<li>Metformin XR 500mg</li>
<li>Diovan 160mg</li>
</p>
